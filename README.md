# File Handling in Python 
In this, I have created two files (i.e. a.txt, b.txt) with some numeric data inside them and then created a third file (c.txt) which contains the Addition, Subtraction and Multiplication of the data present inside the first two files.
## Description 
1. The program creates two files:
   - `a` containing the number `6`
   - `b` containing the number `7`
2. Read the input from both files
3. Performs the following operations:
   - Addition
   - Subtraction
   - Multiplication
4. Stores the results in a third file named `c`
## Project Structure 
```
Filehanding.txt
a.txt
b.txt
c.txt
```
## Concept Used
- File Creation
- File Writing
- File Reading
- Type Conversion (`int()`)
- String Conversion (`str()`)
- Python Context Manager (`with open()`)
## How to Run
```bash
python FileHandling.py
```
